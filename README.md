# F1 Racing Landing Page 🏎️

A beautiful, responsive landing page featuring Formula 1 racing. Built with pure HTML, CSS, and JavaScript — no frameworks or dependencies required.

## Features

- **Hero Section** — Dramatic full-screen hero with animated speed lines and call-to-action buttons
- **Live Stats Bar** — Key F1 statistics displayed with bold typography
- **Teams Section** — Card-based grid showcasing legendary F1 teams with images and details
- **About Section** — Split layout describing the F1 experience with feature highlights
- **Race Schedule** — Upcoming race calendar with flag emojis and circuit details
- **Newsletter CTA** — Email subscription form with sleek dark design
- **Footer** — Multi-column footer with social links, navigation, and branding

## Design

- Dark racing theme with signature F1 red (`#e10600`) accents
- Fully responsive — works on mobile, tablet, and desktop
- Smooth scroll-triggered fade-in animations
- Fixed navigation with blur backdrop and mobile hamburger menu
- CSS-only speed line animations in the hero section

## Getting Started

Simply open `index.html` in any modern web browser:

```bash
open index.html
```

No build tools, bundlers, or package managers required.

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (Intersection Observer, DOM manipulation)
